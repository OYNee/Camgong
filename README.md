## 🤷‍페르소나

20대 중후반 취준생

- 목표는 대기업
- 회사에 따라 다르지만 최종면접까지 간 경험 있음
- 스펙을 샇기 위해 영어공부, 자격증공부와 취업준비를 병행
- 코로나때문에 취업준비도, 공부도 나태해지는 것 같아 불안함
- 밤새는게 잦아 바이오리듬이 엉망이 됨
